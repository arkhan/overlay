Custom Gentoo/Funtoo overlay
=======================

Requisitos
------------
- Gentoo Linux
- app-portage/layman

Modo de Uso
-----
Agregar este *Overlay* con [Layman](http://layman.sourceforge.net/):

    layman -o https://raw.githubusercontent.com/arkhan/overlay/master/repositories.xml -f -a arkhan
